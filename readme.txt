Advanced Google Maps for WordPress is a powerful tool for creating interactive and engaging maps in WordPress websites.

== Description ==
Advanced Google Maps is a plugin for WordPress that provides a range of advanced features for creating customized maps on your website. This plugin provides a simple way to add Google Maps to your WordPress website via a Gutenberg Block. It supports zoom levels, and map types, language and so on.
== Installation ==
Installation is fairly straight forward. Install it from the WordPress plugin repository.
